# Number Guessing Game with GUI

A simple number guessing game where the computer selects a random number between 1 and 100, and the player has to guess it using a graphical interface.

## Features
- The player enters a number and gets feedback ("Too high" or "Too low").
- The game tracks the number of attempts.
- A "Restart Game" button allows the player to start over.
- Simple and beginner-friendly code.

## How to Run
1. Ensure you have Python installed. You can download it from [python.org](https://www.python.org/).
2. Clone this repository or download the files.
3. Run the program using the following command:
   ```bash
   python guess_the_number_gui.py
